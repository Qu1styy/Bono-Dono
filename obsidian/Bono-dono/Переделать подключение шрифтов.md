и подключить следующим образом

```
@font-face {
  font-family: 'MyCustomFont'; /* Имя шрифта для использования в CSS */
  src: url('path/to/font.woff2') format('woff2'),
       url('path/to/font.woff') format('woff');
  font-weight: normal;    /* Начертание (normal, bold и др.) */
  font-style: normal;    /* Стиль (normal, italic и др.) */
}
```
